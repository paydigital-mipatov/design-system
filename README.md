# Design System

Дизайн-система paydigital — токены, компоненты, гайдлайны.

## Структура

```
design-system/
├── tokens/          # Цвета, типографика, отступы, тени
├── components/      # UI-компоненты
├── guidelines/      # Правила использования
└── README.md
```

## Начало работы

1. Клонируй репозиторий
2. Изучи токены в `/tokens`
3. Смотри компоненты в `/components`
